# 🚀 CampusGPT – AI Student Assistant

CampusGPT is a modern AI-powered chatbot website built using HTML, CSS, JavaScript, and Google Gemini API.

This project was created as part of an AI Web Development Internship to explore how AI can be integrated into websites and used for real-time conversations.

## ✨ What This Website Can Do

- Answer user questions using AI
- Have real-time conversations
- Help students with learning and productivity
- Provide a smooth chatbot experience
- Work on desktop and mobile devices

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Google Gemini API
- Fetch API

## 💡 Features

- AI chatbot integration
- Responsive modern UI
- Smooth animations
- Chat bubbles for user and AI
- Loading animation
- Enter key support
- Dark futuristic theme
